Application de gestion de stock réaliser avec Supabase (la base de données), Netlify (site de publication), et HTML CSS
index contion le code de l'application
icon : le logo de l'entreprise
manifest : dit au navigateur que le site est une application installable (nom, icône, couleurs).
sw : met en cache l'application pour qu'elle se charge plus vite et fonctionne partiellement sans internet.

Les fonctionalités de l'application :
✅ Gestion des produits et catégories
✅ Entrées, sorties, ajustements
✅ Blocage des ventes en rupture de stock
✅ Annulation de vente avec mise à jour automatique
✅ Inventaire comparatif avec écarts et marges
✅ Ventes par jour avec montants nets
✅ Alertes stock faible et rupture
✅ Actualisation en temps réel (Supabase)
✅ Installable sur Android et PC
✅ Logo personnalisé "Les Délices de Caro"
